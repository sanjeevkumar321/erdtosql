# ERDtoSQL

A professional, high-performance visual database designer and SQL generator. 

## Features
- **Visual ERD Designer**: Drag-and-drop tables and build relationships.
- **Multi-Dialect Support**: Generate SQL for PostgreSQL, MySQL, and SQLite.
- **Framework Ready**: Export to Prisma Schema in seconds.
- **Visual Customization**: Assign colors to tables for better organization.
- **Instant Export**: Download SQL scripts, JSON schemas, or SVG images of your diagrams.
- **Privacy First**: All data is saved locally in your browser. No login required.

## Technical Details
Built with React, Vite, and Shadcn UI. Uses `@xyflow/react` for the visual canvas.

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`
